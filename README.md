# Res-Fed-ViT: Resilient Federated Vision Transformers for Alzheimer's Disease Prediction

This repository contains the official implementation of **Res-Fed-ViT**, a resilient federated learning framework for **Alzheimer's disease prediction from brain MRI images**.

The framework combines **Federated Learning (FL)**, **Vision Transformer-based architectures**, **CNN-ViT hybrid models**, and **adversarial robustness strategies** to improve model reliability under heterogeneous data distributions, limited data diversity, and adversarial perturbations.

The repository implements robust FL-ViT variants using different federated learning strategies, backbone architectures, and robustness mechanisms.
